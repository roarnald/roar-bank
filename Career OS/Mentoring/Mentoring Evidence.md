## Direct Mentoring Relationships

### Jacky — Full Stack Developer (2025)
- **Context**: Jacky had never built a server-side application. Ronald assigned him as the lead on Lotto6D KJW NodeJS Microservice.
- **Approach**: Broke down deliverables into bite-sized steps; provided working code samples from own experience; collaborated on blockers; allowed Jacky to own the solution.
- **Outcome**: Production microservice delivered; Jacky presented findings at Frontend Sharing and credited Ronald publicly.
- **Impact**: Lifted overseas team dependency; extended full-stack capability to a second developer.
- [[2025-01-lotto6d-nodejs-mentoring]]

### UIUX Team Developers (2023)
- **Context**: 4 developers with legacy HTML/CSS/JS background joined the React/Web Team.
- **Approach**: Paired each with a seasoned React developer; individual consultations; managed transition anxieties; maintained open communication about expectations.
- **Outcome**: All developers successfully transitioned; working independently in React/Web Team workflows.
- [[2023-01-uiux-team-merger]]

### Junior Developer Enlightenment (2021)
- **Context**: Code review mentoring; identified a junior developer using an inefficient JavaScript function.
- **Approach**: Taught to convert to lower-level for-loop; shared reasoning rooted in low-level programming background (Assembly, C).
- **Outcome**: Junior gained understanding of performance trade-offs; team discussion was collaborative, not prescriptive.
- Evidence: 2021 Appraisal — Creativity & Problem Solving section
