# Organizational Leadership

## Definition
Designing team structures, driving systemic change, establishing processes and culture that outlast any individual, and making decisions that shape how an organisation operates.

## Key Demonstrations

### Team Restructuring
- **UIUX Team Merger (2023)**: Identified a structural inefficiency (UIUX team maintaining legacy stack, creating debt for React team); built the business case; secured management buy-in; executed the merger. Result: freed 4 headcounts, unified Web Team of 30. — [[2023-01-uiux-team-merger]]
- **IC Role System (2024–2025)**: Redesigned how leadership roles were distributed — from centralised (leads do everything) to distributed (seniors own event ICs with 2ICs). — [[2024-01-leadership-decentralisation]]
- **Team Specialisation Proposal (2025)**: Proposed Game Engine Team and NodeJS/Fullstack Team as dedicated sub-teams with identified project backlogs — structured path to specialisation at scale.

### Process Design
- **Architecture Review Process (2024)**: Proposed and implemented SA Review gate for new frontend projects. Prevented technical debt accumulation at project inception. Applied to Rollball, PNL, credit draw.
- **Urgency Tagging (2021)**: Workflow innovation solving code review tension — designed, implemented, followed up at 2 months. — [[2021-11-urgency-tagging-workflow]]
- **Anonymous Internal Feedback (2023+)**: Added objectivity to appraisal input cycles.
- **Node Upgrade Orchestration (2023)**: Coordinated upgrade of entire repository portfolio.

### Culture Building
- **Frontend Sharing (2022–2025)**: 4-year programme; started, scaled, delegated, and extended cross-company. — [[2022-01-frontend-sharing-program]]
- **Buddy System (2022+)**: Newcomer integration and first leadership step for seniors.
- **Frontend Spotlight (2023+)**: Anonymous Q&A event to build interpersonal connections.
- **State of Merquri / Frontend Meet**: Kickstart events communicating team direction changes.

### Span of Influence
- Led a team that grew from a small group to **30 members** over 6 years.
- Reached beyond the Frontend team: UIUX merger, cross-company Aquarix sharing, Action Plan group (Architects + CTO).
